Quick Start

A jump-start guide to getting going with Roboflow.
Watch the Getting Started video for an overview of the platform.

Guided Tutorial

When you create your first project, you will be guided through an in app tutorial. Just click "Create Project" to get started. No data, no worries? You can use our sample dataset with some images of chess pieces to use with the tutorial.
Or, follow our getting started guide.

Quick Start Repo

GitHub - roboflow/quickstart-python
GitHub
  1. 1.
    Install all the dependencies you need to get started with Roboflow's Python package;
  2. 2.
    Get you set up running live inference on object detection models with your webcam in a Jupter notebook;
  3. 3.
    Guide you through using object detection, semantic segmentation, and image classification models in Roboflow projects and;
  4. 4.
    Provide you with more resources to start training your own model.
